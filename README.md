## 概要

CCD/IMG/SUBのCCDをCUEに変換するスクリプトです。  
https://github.com/Meetsch/ccd2cue を参考にしています。  
CCDのファイルフォーマットは非公開のようなので変換の正しさは保証されません。

## インストール

```
# git clone https://github.com/yuichietsu/ccd2cue/
# cd ccd2cue
# composer install
```

## 使い方

```
# composer run convert -- /mnt/d/tmp/game.ccd
```
